# LinGOAT Chrome Extension

Learn any language while doomscrolling.

It is a subtitle service that allows you to watch any video in any language and transcribes + translates it in real time. Providing extra features such as romanization, bilingual support, in-built notes and vocab list, and your personal AI tutor all for free, all in one.

Become a lingo GOAT with LinGOAT 🐐


## Features

- 🎙 **Live subtitles** video subtitles are scraped and translated. If no subtitles are available, LinGOAT live captures tab audio output and translates on the spot with Groq Whisper API
- 🗾 **3 line subtitles**: target language / romanization / translation
- 🖱 **Click any word** popup with definition, reading, examples, grammar notes
- 🔀 **Dual native language toggle** switch between two translations instantly (e.g. English ↔ Chinese translations for Japanese)
- 📓 **Floating notes panel** draggable, minimizable, persistent
- ⭐ **Star vocabulary** to highlight important words
- 🔍 **Word detail view** full dictionary entry with examples
