How it works
1

Set your languages

Click the LinGOAT icon in your toolbar, pick the language you're learning and your native language.

2

Play any video

Go to YouTube, Crunchyroll, or any site with video. If the site has subtitles, LinGOAT will automatically scrape and translate them, no extra steps needed!

3

No subtitles? Start Recording

If the video has no subtitle service, click the LinGOAT icon and hit "Start Capture" to record the tab's audio and transcribe it with AI. Pause the video frequently to learn at your own pace.

4

Subtitles appear

A floating subtitle bar shows the original text, readings (e.g. furigana), and your translation live. Click any word for a dictionary lookup!

🎤

About "Start Capture" and microphone permission

You only need "Start Capture" if the video doesn't already have subtitles. If the site provides subtitles (YouTube CC, Crunchyroll, Netflix, etc.), LinGOAT automatically scrapes and translates them, no recording needed!

When you do use Start Capture, Chrome may show a permission prompt. This is normal. LinGOAT captures your tab audio only (not your physical microphone) and sends it to Groq's Whisper API for transcription. Your Groq API key is required for this feature.

Sometimes, the found subtitles are already translated, in such instances, toggle "Use audio instead of page CC" in the popup menu to see the original spoken language.

Caution: Translation is not 100% accurate. If it's not what you expect, ask your AI tutor for help!

Troubleshooting
Subtitles aren't showing up
Reload the page. LinGOAT starts when the page loads, so a fresh load is the first fix to try. Then click ▶ Show Subtitle Bar in the popup to bring it back if it was hidden. If it's still not working, check if your Groq API is inputted in the popup.
The subtitle bar disappeared
Click the LinGOAT icon in your toolbar and hit ▶ Show Subtitle Bar. It hides automatically when you close it and won't reappear until you ask for it.
Translation is in the wrong language
Open the popup, check that Language you're learning and Primary native language are set correctly, then hit Save Settings. If the page CC is already in your native language, toggle Use audio instead of page CC.
Audio capture says "listening" but never transcribes
Make sure your Groq API key is entered in the popup. You can get a free key at console.groq.com → API Keys. Also check you're on the right tab since Start Capture only captures the tab it was started on.
Word clicks / AI Tutor aren't responding
Reload the page. If it keeps happening, try disabling and re-enabling the extension in chrome://extensions.
Subtitles show on every page, even ones without video
Turn off Auto start on page load in the popup settings. The bar will only appear when you manually click ▶ Show Subtitle Bar.
Translations aren't accurate or feel cut off mid-sentence
Pause the video at the end of each sentence and wait a moment for the transcription and translation to finish before playing again. Groq Whisper processes audio in chunks, so if you keep playing while it's still working, the next chunk starts before the current sentence is complete, and the meaning can be misread. General flow: hear a line → pause → read + learn → play.
Furigana / readings are missing or wrong
This is handled by the Groq AI, so if you're out of free-tier requests, readings fall back to basic mode with no furigana. Check your usage at console.groq.com.
